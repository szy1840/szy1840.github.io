Overview

The site you are reading. It is built with Eleventy and a fully custom design system, with no theme and no client framework. Technical notes can carry both a Chinese and an English version with an inline language toggle, and writing lives in its own quieter section.

Why I built it

I wanted something that felt like mine rather than a portfolio template, and that stayed simple to maintain. Markdown in, static HTML out.

Notes

Type is set in Fraunces and Inter. Everything is one stylesheet and a small amount of JavaScript for the theme toggle, the mobile menu, and the table of contents. If you are curious how a page is put together, the source is linked above.
